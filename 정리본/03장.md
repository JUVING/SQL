# *전체 데이터베이스 구성도
- 데이터베이스 모델링의 결과로 다음과 같이 구성이 완료되었다.
- 데이터는 행 단위로 입력 받는다.
- 행(가로) 데이터 갯수가 데이터의 갯수이다.
- PK(기본키) : 행을 구별하는 유일한 값
	- 기본키는 *중복으로 할 수 없다.
	- 기본키의 *데이터는 비어 있을 수 없다.
	- 기본키는 제작자가 지정 가능하다.
	- 기본키는 열에 지정 가능하며 테이블 당 하나만 지정 가능하다.
- 데이터 형식 : 열에 저장할 데이터의 형식
